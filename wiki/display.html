Display System

Direct Display


display("Hello, World!")  # String output
display(42)              # Number output

Variable Display

Uses %var[n] format where n is the variable position (1-based)


x = 42
display("%var1", x)  # Shows value of x

Multiple Variable Display


age = 25
name = "John"
display("Age: %var1, Name: %var2", age, name)
