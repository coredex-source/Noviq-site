Arithmetic Operations

Supported Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • // Floor Division
  • % Modulus
  • ** Exponentiation

Examples


sum = 5 + 3         # Addition
diff = 10 - 4       # Subtraction
prod = 6 * 2        # Multiplication
quot = 15 / 3       # Division
floor_div = 17 // 5 # Floor Division (Results in 3)
remainder = 17 % 5  # Modulus (Results in 2)
power = 2 ** 3     # Exponentiation (Results in 8)

Rules

  • Operands can be numbers or variables
  • Operations between different types (int/float) result in float
  • Division always produces float results
  • Division by zero produces an error
