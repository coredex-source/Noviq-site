String Formatting System

Basic Format

Noviq supports dynamic string creation using variables:


name = "John"
greeting = ("%var1, welcome!", name)

Multiple Variables

Combine multiple variables in any order:


name = "John"
age = 25
info = ("%var1 is %var2 years old", name, age)
