Logical Operations

Supported Operators

AND or && Logical AND
OR or || Logical OR
NOT or ! Logical NOT
>, < Greater/Less than
>=, <= Greater/Less than or equal
== Equal to

Boolean Operations


# Basic logical operations
result = true AND false    # Results in false
result = true && false     # Alternative syntax
result = true OR false     # Results in true
result = NOT true          # Results in false

Comparison Operations


# Number comparisons
x = 5
y = 10
isLess = x < y          # true
isEqual = x == y        # false
isGreater = x > y      # false

Type Conversion Rules

  • Numbers: 0 is false, any other value is true
  • Strings: Empty string is false, non-empty is true
  • Booleans: Used directly
  • Results are always boolean type
