Variable System

Noviq supports four primary variable types:

Integer Variables

Whole numbers (positive or negative)


count = 42  # Integer assignment

String Variables

Text enclosed in quotes


name = "John"  # String with double quotes
message = 'Hello'  # String with single quotes

Float Variables

Decimal numbers


price = 42.99  # Float assignment

Boolean Variables

True/false values


isValid = true   # Boolean true
isDone = false   # Boolean false
